# 🎮 Terminal Games — 终端游戏集合

纯 Bash 编写的终端游戏，支持彩色 ANSI 渲染和方向键控制。

## 启动

```bash
# 打开游戏启动器（推荐）
bash games/launcher.sh

# 直接运行单个游戏
bash games/snake.sh
bash games/2048.sh
bash games/minesweeper.sh
```

## 游戏列表

### 🐍 贪吃蛇 (Snake)
| 按键 | 功能 |
|------|------|
| `↑↓←→` | 控制方向 |
| `p` | 暂停/继续 |
| `r` | 重新开始 |
| `q` | 退出 |

- 吃到食物蛇身变长，速度逐渐加快
- 最高分自动保存到 `/tmp/.snake_hs`

---

### 🔢 2048
| 按键 | 功能 |
|------|------|
| `↑↓←→` | 移动合并方向 |
| `r` | 重新开始 |
| `q` | 退出 |

- 合并相同数字块，达到 **2048** 获胜
- 达到 2048 后可继续挑战更高分
- 最佳分数保存到 `/tmp/.2048_best`

---

### 💣 扫雷 (Minesweeper)
| 按键 | 功能 |
|------|------|
| `↑↓←→` | 移动光标 |
| `Space/Enter` | 翻开格子 |
| `f` | 插旗/取消旗帜 |
| `r` | 重新开始 |
| `q` | 退出 |

- 三档难度：简单 9×9 / 中等 16×16 / 困难 16×30
- **首次点击保证安全**（点击位置及周围不会有地雷）
- 空白区域自动展开

## 系统要求

- Bash 4.0+（Ubuntu 默认满足）
- 终端支持 256 色 ANSI（iTerm2、Windows Terminal、Alacritty 等均支持）
- 终端窗口建议大小：`120 × 40` 或以上
